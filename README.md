Smart Examination Learning System
The Smart Examination Learning System (SELS) is a web-based application designed to manage online examinations and learning processes efficiently. The system provides separate interfaces for administrators and students, enabling smooth exam management and interaction.
Technologies Used:
Frontend: HTML, CSS, JavaScript (React)
Backend: Java (Spring Boot / Core Java)

Features:
User Authentication (Login System)
Admin Dashboard
Student Dashboard
Exam Creation and Management
Question Handling
Result Processing
Responsive User Interface
